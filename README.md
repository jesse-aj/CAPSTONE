🌦️ Weather Now App

A simple and responsive weather application built with React and Tailwind CSS. It allows users to search for any city and view real-time weather data, temperature conversions (°C/°F), and forecasts — all with light/dark mode support.

🚀 Features

🌍 Search weather by city name

🌡️ Switch between Celsius and Fahrenheit

🔄 Refresh button for live updates

🕶️ Light & dark mode toggle

📅 5-day weather forecast

💨 Displays temperature, humidity, wind speed, and conditions

🛠️ Tech Stack

React.js (Frontend)

Tailwind CSS (Styling)

OpenWeatherMap API (Weather Data)

⚙️ Installation

Clone the repository:

git clone https://github.com/jesse-aj/CAPSTONE/weather-now.git


Navigate to the project folder:

cd weather-app


Install dependencies:

npm install


Create a .env file in the root directory and add your API key:

VITE_WEATHER_API_KEY=your_api_key_here


Run the development server:

npm run dev

🧠 Usage

Type any city name into the search bar.

Toggle between °C and °F using the “Switch Units” button.

Toggle between light and dark mode.

Click “Refresh” to get updated weather info.
